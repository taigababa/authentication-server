第1条（適用）
1. 本規約は、本サービスに関わる一切の関係に適用されます。
2. 当社が本サービスに関して、本規約のほかガイドライン等を定めた場合には、当該ガイドライン等も本規約の一部を構成するものとします。

第2条（利用資格）
1. 利用者は、TikTok の利用規約および API 利用ポリシーに従って本サービスを利用しなければなりません。
2. 未成年の利用者が本サービスを利用する場合、親権者等の法定代理人の同意を得たものとみなします。

第3条（利用登録）
1. 利用者は、本サービスを利用するにあたり、所定の方法により TikTok アカウント連携等、必要な情報の登録を行うものとします。
2. 利用者は、登録に際して真実かつ正確な情報を提供し、内容に変更が生じた場合は速やかに更新するものとします。

第4条（取得・利用する情報）
1. 当社は、TikTok API を通じて取得する以下の情報を、本サービスの提供に必要な範囲で利用します。
   (1) アカウント情報（フォロワー数、フォロー数、動画数、いいね数 等）
   (2) 投稿情報（動画 ID、投稿日時、サムネイル、キャプション、各種エンゲージメントデータ 等）
   (3) インサイト情報（動画視聴数、ユニーク視聴数、視聴完了率、平均視聴時間、新規フォロワー数 等）
   (4) 利用状況に応じたログ情報や端末情報等の技術情報
2. 当社は、取得した情報を本サービスの提供、改善、サポート、セキュリティ確保および統計分析（個人を特定できない形式に限る）に利用します。
3. 個人情報の取扱いに関する詳細は、当社のプライバシーポリシー（https://auth.yolo-shonan.com/privacy-policy）をご参照ください。

第5条（利用者の責任）
1. 利用者は、本サービスの利用および本サービスを通じて行った行為について、一切の責任を負うものとします。
2. 利用者は、自己の管理において適切なセキュリティ対策を講じ、本サービスで得た情報を厳重に取り扱うものとします。

第6条（禁止事項）
利用者は、本サービスの利用に際し、以下の行為をしてはなりません。
1. TikTok の利用規約または関連ポリシーに違反する行為
2. 不正アクセス、リバースエンジニアリング、解析その他本サービスのセキュリティを侵害する行為
3. 本サービスで取得した情報を利用者本人以外の第三者に無断で提供・販売する行為
4. 法令、公序良俗、本規約、または当社が不適切と判断する行為
5. 当社または第三者の権利・利益を侵害する行為

第7条（サービスの提供・変更・中断）
1. 当社は、以下の事由がある場合には、利用者への事前通知なく本サービスの全部または一部を変更・中断・終了できるものとします。
   (1) TikTok API の仕様変更や提供停止が生じた場合
   (2) システム保守、点検、更新をやむを得ず実施する場合
   (3) 天災地変、停電、通信障害等の不可抗力が発生した場合
   (4) その他、当社が運営上必要と判断した場合
2. 当社は、前項に基づく措置により利用者に生じた損害について、一切の責任を負いません。

1. 当社は、本サービスおよび本サービスを通じて提供される情報について、正確性、完全性、有用性等を保証するものではありません。
2. 利用者と TikTok または第三者との間で生じた紛争について、当社は一切責任を負いません。
3. 利用者が本サービスを利用したことまたは利用できなかったことに起因して生じたいかなる損害に対しても、当社は責任を負いません。

第9条（知的財産権）
本サービスに関する著作権、商標権その他一切の知的財産権は、当社または正当な権利者に帰属します。利用者は、当該権利を侵害する行為を行ってはなりません。

当社は、利用者が本規約に違反した場合、または当社が本サービスの利用を不適切と判断した場合には、事前通知なく利用停止、利用制限または登録抹消の措置を講ずることができます。
